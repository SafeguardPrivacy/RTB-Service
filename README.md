# Safeguard✓Privacy OpenRTB Service

In order to run this API, you will need a Safeguard✓Privacy API Token.

## Installation

```bash
$ npm install
$ cp .env.example .env
```

## Update .env file

Add your SafeguardPrivacy API Token to the .env file along with your redis credentials.

## Running the app

```bash
# development mode
$ npm run dev

# production mode
$ npm run build
$ npm run start:prod
```
